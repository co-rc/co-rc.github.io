# Resources

## Firmware Files

Binary files (firmware) resulting from experiments with a custom CircuitPython build for the XIAO nRF52840 Plus.

Detailed motivation and decisions can be found in [ADR 0010: Custom CircuitPython Build for XIAO nRF52840 Plus](../architecture/adr/0010-custom-circuitpython.md).

`CP (name='circuitpython', version=(10, 0, 3, ''), _machine='Seeed XIAO nRF52840 Sense with nRF52840', _mpy=774, _build='Seeed_XIAO_nRF52840_Sense')`

- [XIAO_nRF52840_full.uf2](XIAO_nRF52840_full.uf2) – Full firmware version.
- [XIAO_nRF52840_custom.uf2](XIAO_nRF52840_custom.uf2) – Custom version (optimized).
